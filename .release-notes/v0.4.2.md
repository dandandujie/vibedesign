## v0.4.2

生成过程现在会展示真实、可验证的 Agent 运行状态，让等待不再像“卡住”。

### Agent 运行状态
- 显示准备设计上下文、等待模型、生成设计内容、整理并更新画布等真实阶段。
- 状态条显示实时耗时，点击可展开完整步骤；完成、停止和失败均有明确结果。
- 服务端通过 SSE 主动发送状态和 10 秒心跳，不再由前端根据 HTML 输出猜测 Agent 是否仍在工作。
- 模型响应超过 20 秒时给出慢响应提示；长时间未收到事件时提示用户继续等待或停止重试。

### 体验与无障碍
- 首次生成时，画布显示清晰的 Agent 等待卡片，预览生成后自动替换。
- 完整支持中文和英文界面。
- 遵循系统“减少动态效果”设置，关闭非必要脉冲与旋转动画。
