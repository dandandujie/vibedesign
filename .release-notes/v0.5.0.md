## v0.5.0

大版本：Vibedesign 与主流 coding agent 全面打通，并从单页迭代进化为成系统的多页原型设计。

### Agent 打通（/design）

- 新增 MCP stdio server（`vd_design` 等 5 个工具）与 headless 生成 API，一条命令接入 11 个 agent：`npm run install:agents`（Claude Code / Codex / Cursor / OpenCode / Hermes / Grok Build / Antigravity / Kimi Code / Qoder / Trae / Pi）。
- **设置 → Agent 打通**可视化面板：各 agent 状态实时可见（CLI 探测 / MCP 配置 / /design 技能），勾选即打通、取消即移除。
- 桌面版用应用自身二进制作为 MCP 进程，安装版用户也能真正连上。
- 在终端里说「/design」即可驱动 Vibedesign 生成设计，画布链接实时可看可改，定稿后由 agent 把 HTML 落回代码库。

### 更新提醒

- 检测到新版本时自动弹出居中更新弹窗（背景模糊），可一键自动更新并重启；同一版本只提醒一次，更新日志按钮保持高亮。

### 站点原型与原型管理

- 新交付协议 vdsite：一次生成多页互联原型（共享样式、页面互联、site.json 清单），配套 site-prototype 技能按「sitemap 确认 → 全站骨架 → 逐页精修」分层工作。
- 概览画布：按页数自适应宫格（1 全屏 / 2 半屏可切换方向 / 3 列 / 4 宫格 / 6 / 9 / 更多滚动），手机模式下每页以真机壳展示。
- 页面管理：增删改排序页面；版本管理弹窗：搜索 / 预览（多页平铺）/ 恢复 / 逐版本导出。
- 站点编辑接入完整编辑管线：选元素改样式即改即存，写回站点文件。
- 新技能 mobile-flow：App 流程线框板（第 35 个技能）。

### 画布与导出

- Web / 手机 / App 文字设备按钮 + 六种真机壳（iPhone 深浅 / Android / iPhone SE / iPad / 简约）。
- 视频导出可选帧率 / 尺寸 / 格式，渲染进度逐帧可见；动效教学资产扩充（25+ 转场 + 8 种视觉风格）。
- 站点 ZIP 导出（全部页面 + SITE-MANIFEST）。

### 修复

- 折叠侧栏后画布布局崩坏；多页编辑渲染丢样式；弹层被屏幕边缘截断；新会话改为当前窗口打开（保留画布、清空对话）。
