## v0.4.0

参考并学习开源项目 [open-design](https://github.com/nexu-io/open-design) 的能力，对「设计大脑 / Live Artifacts / 幻灯 / 输入表单」做了一轮完整升级——不是简单接入，而是把这些能力做到位、并按 Vibedesign 的运行模型落地。

### 设计大脑（质量核心）
- **craft 设计规则完整版**：排版 / 色彩 / 可访问性 / 交互状态 / 动效 / i18n（RTL）/ 表单校验等 12 个模块换成完整版（约 5.9× 内容），并补齐此前缺失的 animation-discipline / form-validation / rtl-and-bidi / editorial 排版层级——技能生成时注入的设计质量指导大幅增强。
- **内置设计系统品牌保真**：为内置设计系统补上机器可读的 `tokens.css`（`:root` 令牌契约），生成时原样注入、只允许 `var(--*)`、禁止裸 hex——最强的品牌一致性保障被真正激活。
- **设计系统扩到 159 个品牌** + 23 个带**中文本地化**（中文模式下优先用中文设计规范）。

### 平台能力
- **PNG / PDF 像素级导出**：无头 Chromium 真渲染，正确渲染中文字体与 WebGL（客户端截图做不到）。
- **copy starter 起手模板**：web 原型 / 移动 App / 杂志幻灯 / 看板 / 评审报告等技能内置起手脚手架，一致性与质量地板更高。
- **多文件 + preview.entry**：可产出 `index.html + styles.css + app.js` 的多文件设计，服务端按真实 URL 提供、兄弟文件相对解析。
- **多窗口演讲者视图**：幻灯可开控制窗（当前 / 下一张 + 逐字稿 + 计时）+ 观众窗，跨窗同步。
- CDN 字体 / WebGL 在预览中开箱可用。

### 幻灯（HTML PPT）
- 内置完整 **html-ppt 运行时**（36 主题 + 动画 + 画布特效 + 键盘导航 + S 键演讲者模式），咨询汇报幻灯通过 `/api/deck-assets` 引用，落到真实 URL 正常翻页 / 主题 / 进度。

### Live Artifacts（可刷新数据）
- **安全加固**：刷新数据强制边界校验（深度 / 数组 / 体积 / 拒绝凭据类字段）；`http_json` 加超时 + 响应体上限 + 拦截私网 / 回环 / 云元数据地址（防 SSRF）。
- **多数据源**：新增只读**连接器**（GitHub 仓库 / Hacker News / 加密价格）+ 沙箱**本地 JSON 源**；输出 **transform**（大数组自动折成表格 / 指标摘要）；**溯源**面板（数据来自哪个源、何时刷新）。

### 输入表单（inputs）
- 生成前的关键输入表单从 4 个技能扩到 **15 个**（原型 / 落地页 / 看板 / 移动 / 营销 / 文档 / 幻灯）。
- 新增 **数字步进器** 与 **布尔开关** 控件；技能声明的可调参数（密度 / 间距 / 强调色…）**自动**变成画布上的实时滑块。

### 其它
- 内联编辑、视觉标注、换肤、Markdown 文档类型、原型设备视图（web / 移动端 / 移动应用）等此前版本的能力持续可用。
