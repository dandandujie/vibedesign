## v0.4.3

本次更新修复 macOS 窗口标题栏遮挡，并加入可从当前设计开启独立短上下文会话的新窗口能力。

### macOS 窗口适配
- 为左上角红、黄、绿窗口按钮设置稳定位置，并给编辑器侧栏预留安全区。
- 侧栏折叠后，画布标题栏与展开按钮也会自动避开窗口按钮。
- 首页和设计系统设置页同步适配，不再出现文字或操作按钮重叠。

### 同项目新会话窗口
- 首页项目菜单和编辑器项目菜单新增「在新窗口开启新会话」。
- 新会话保留当前画布、版本、评论与设计系统，但不携带旧聊天历史，降低长上下文导致 API 超限或调用失败的风险。
- 第一条新指令会自动附带当前画布快照，可以从现有设计继续工作。
- 每个会话使用独立数据记录，避免多个窗口并发保存时互相覆盖。
