## v0.3.1

### 新功能
- **.fig / .pen 文件导入**：采用 open-pencil 的 Kiwi codec 方案（fig-kiwi + kiwi-schema，deflate/zstd 双解压），完全在本地解析，不上传。自动提取色板、字体/字号、圆角、组件命名与文案样本，作为设计上下文注入生成。入口：编辑器 ＋ 菜单「Upload .fig / .pen file」、Design system 创建页。

### 修复与优化
- 所有弹出卡片/菜单（模型选择、＋ 菜单、更新日志、Share、项目行菜单等）自动检测视口边界，不再被遮挡或截断
- 更新日志卡片现在显示每个版本的真实更新内容（本文件机制 + 自动生成兜底），并清理了 markdown 噪音
- 旧版本（v0.1.0–v0.3.0）的发布说明已补全
