# Skill starter seeds

Prebuilt starter templates a skill hands the model as its "copy this and modify"
starting point (token system / device frame / deck runtime already wired). When a
skill has a `<skillId>.html` seed here, the brain injects it into the system
prompt for that skill.

Seeds are adapted from open-design's `design-templates/*/assets/template.html`
(Apache-2.0 © the Open Design authors).
